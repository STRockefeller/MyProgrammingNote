## 用gist架設靜態網站



### 使用bl.ocks.org

https://bl.ocks.org/

> For example, if your Gist URL is:
>
> ```
> https://gist.github.com/mbostock/1353700
> ```
>
> Replace “gist.github.com” with “bl.ocks.org” in the URL to view it:
>
> ```
> https://bl.ocks.org/mbostock/1353700
> ```
