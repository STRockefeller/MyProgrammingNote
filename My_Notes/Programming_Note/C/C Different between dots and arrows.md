倒是早期C語言的邏輯有些用現在的角度去看會覺得十分不可思議

**Conclusion**

早期的C語言中`->`代表的是位址的偏移

在現版本的C和C++中`a->b`等同於`(*a).b`

例如

```C
Struct A{
    int b;
    int c;
};
A *a;
a->b = 10;
(*a).b = 10; //跟上面那行是一樣的動作。
```
